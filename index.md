## ❓ Common Questions

Can I change the colors of the dashboard?
Yes. Use the standard Excel formatting tools to change cell colors. This will not affect the functions.

Do I need to enable macros?
No. The file uses standard Excel functions. You can open it with macros disabled.

Will it work on a Mac?
The file is optimized for Windows. While it should open on a Mac, some layout features might look different.

How do I add a new book to the reading list?
Go to the Reading List tab. Type the book title in the next empty row. The dashboard will include it in your total count.

What if the file runs slow?
If you add hundreds of lines, Excel might take a moment to calculate. This usually resolves in a few seconds.

Is my data private?
You own the file. The data stays on your hard drive unless you choose to share the file with others. No information travels to a web server.
